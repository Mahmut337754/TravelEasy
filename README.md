# TravelEasy - Reisboeking Systeem

Een modern webbased reisboeking systeem gebouwd met React, TypeScript en CSS.

## Gebruikersrollen
- **Reisadviseur**: Boekingen beheren, klanten adviseren
- **Financieel medewerker**: Financiële transacties en rapportages
- **Manager**: Managementinformatie en overzichten
- **Administrator**: Systeembeheer en gebruikersbeheer

## Prijsstructuur
- Baby's (0-2 jaar): Gratis
- Kinderen (2-12 jaar): 10% korting
- Volwassenen: Volledige prijs
- Administratiekosten: €25 per boeking
- SGR-kosten: €5 per persoon
- Annuleringskosten: 5% van reissom

## Tech Stack
- React 18+
- TypeScript
- CSS3 (Responsive design)
- Vite (Build tool)

## Project Structuur
```
src/
├── components/     # Herbruikbare UI componenten
├── pages/         # Pagina componenten
├── services/      # API services
├── types/         # TypeScript types
├── utils/         # Helper functies
├── hooks/         # Custom React hooks
└── styles/        # Global styles
```

## Getting Started

```bash
npm install
npm run dev
```
