# TravelEasy - Modern Reissite Project

## Overzicht
**TravelEasy** is een innovatieve en gebruiksvriendelijke reiswebsite gericht op het verbeteren van de ervaring van reizigers in Nederland. Deze applicatie combineert moderne technologieën zoals **React**, **Laravel**, **MVC-architectuur**, en **AI-gebaseerde aanbevelingen** om een complete, interactieve en veilige reiservaring te bieden.  

De applicatie ondersteunt:
- Online zoeken naar reizen, vluchten, en accommodaties
- AI-gestuurde aanbevelingen op basis van gebruikersvoorkeuren
- Responsive en moderne interface voor desktop en mobiel

---

## Features

### ✈️ Reiszoekfunctionaliteit
- Snel zoeken op bestemming, datum en prijs
- Filteropties voor vliegtickets, hotels en complete pakketreizen
- Real-time prijsupdates en aanbiedingen

### 💻 Modern Web Design
- Gebouwd met **React + Tailwind CSS**
- Mobile-first responsive design
- Animaties en interactieve componenten voor een premium look & feel

### 🔒 Beveiliging & Authentificatie
- Laravel Sanctum voor veilige login & token-gebaseerde auth
- Role-based access control (Admin, User)
- Versleutelde wachtwoorden en session management

### 🗂 Overige Functionaliteiten
- CRUD voor reizen, bestemmingen en gebruikersbeheer
- Geavanceerde database via **MySQL** met genormaliseerde structuur (3NF)
- Stored procedures en joins voor efficiënte data queries
- Logging van systeemacties en gebruik van PSR-12 codeconventie
